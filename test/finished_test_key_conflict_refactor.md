# 按键冲突检测重构测试清单

## 测试目标
验证重构后的按键冲突检测逻辑是否符合以下规则：
1. 作战按键 + 快捷按键 + 启用/禁用热键快捷键 互相检测冲突
2. 卫戍协议按键 + 启用/禁用热键快捷键 互相检测冲突
3. 卫戍协议按键不与作战按键和快捷按键检测冲突
