# 标签页热键切换功能测试清单

## 测试目标
验证标签页切换时热键分组的正确启用/禁用逻辑

---

## 测试用例

### 测试1：默认状态热键启用
**前置条件**：小助手启动，默认在"作战按键"标签页
**操作步骤**：
- [x] 1. 确认小助手启动后托盘图标显示"热键已启用"
- [x] 2. 进入游戏，测试作战按键（如按下F键暂停）是否生效
- [x] 3. 测试快捷按键（如有绑定的话）是否生效

**预期结果**：
- 作战按键和快捷按键相关热键均可用
- 卫戍协议热键不可用

---

### 测试2：切换到卫戍协议标签页
**前置条件**：当前在"作战按键"或"快捷按键"标签页，热键已启用
**操作步骤**：
- [x] 1. 点击"卫戍协议"标签
- [x] 2. 进入游戏，测试卫戍协议按键是否生效
- [x] 3. 测试作战按键是否不再生效
- [x] 4. 测试快捷按键是否不再生效

**预期结果**：
- 卫戍协议热键可用
- 作战按键和快捷按键热键被禁用

---

### 测试3：切换到其他设置标签页
**前置条件**：当前在任意功能性标签页，热键已启用
**操作步骤**：
- [x] 1. 记录当前可用的热键组（作战/快捷 或 卫戍协议）
- [x] 2. 点击"其他设置"标签
- [x] 3. 进入游戏，测试之前可用的热键是否仍然生效

**预期结果**：
- 切换到"其他设置"不改变当前热键状态
- 之前启用的热键组仍然可用

---

### 测试4：禁用热键状态下切换标签页
**前置条件**：热键处于禁用状态（托盘显示"热键已禁用"）
**操作步骤**：
- [x] 1. 先切换到"作战按键"标签页
- [x] 2. 禁用热键（通过托盘菜单或快捷键）
- [x] 3. 切换到"卫戍协议"标签页
- [x] 4. 进入游戏，测试任何热键是否都不生效
- [x] 5. 切换回"作战按键"标签页
- [x] 6. 确认热键仍然不生效

**预期结果**：
- 禁用状态下，切换标签页不会启用任何热键
- 所有热键保持禁用状态

---

### 测试5：从禁用恢复时根据最后标签页启用热键
**前置条件**：热键处于禁用状态
**操作步骤**：
- [x] 1. 禁用热键
- [x] 2. 切换到"卫戍协议"标签页
- [x] 3. 启用热键（通过托盘菜单或快捷键）
- [x] 4. 进入游戏，测试卫戍协议热键是否生效
- [x] 5. 测试作战按键是否不生效

**预期结果**：
- 恢复热键时，根据最后选中的功能性标签页（卫戍协议）启用对应热键组
- 卫戍协议热键可用，作战/快捷热键不可用

---

### 测试6："其他设置"不覆盖最后标签页记录
**前置条件**：热键处于禁用状态
**操作步骤**：
- [x] 1. 禁用热键
- [x] 2. 切换到"作战按键"标签页
- [x] 3. 切换到"其他设置"标签页
- [x] 4. 启用热键
- [x] 5. 进入游戏，测试作战按键是否生效（而非卫戍协议）

**预期结果**：
- 经过"其他设置"后，最后记录的功能性标签页仍是"作战按键"
- 恢复热键时启用作战/快捷热键组

---

### 测试7：快捷按键与作战按键同时启用
**前置条件**：当前在"快捷按键"标签页，热键已启用
**操作步骤**：
- [x] 1. 点击"快捷按键"标签
- [x] 2. 进入游戏，测试快捷按键是否生效
- [x] 3. 测试作战按键是否也生效

**预期结果**：
- "快捷按键"和"作战按键"标签页同时启用作战按键和快捷按键热键组
- 两个组的热键都可用
